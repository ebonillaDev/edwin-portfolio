# Edwin Bonilla | Software Engineer Portfolio

A minimalist, data-driven professional portfolio built with the latest web technologies. This project is designed to showcase engineering experience, AI certifications, and technical skills through a clean, performant interface.

## 🚀 Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Icons & Fonts:** Geist Sans & Mono

## 🏗️ Architecture

This portfolio follows a **Data-Driven Design** pattern. All content is decoupled from the UI components and managed within a centralized data structure (`src/data/resume.ts`). 

- **Separation of Concerns:** UI components handle the rendering logic, while the resume data remains the "Single Source of Truth."
- **Performance:** Leveraging Next.js Server Components for optimized rendering and minimal client-side JavaScript.
- **Type Safety:** Full TypeScript implementation for robust data handling across the application.
